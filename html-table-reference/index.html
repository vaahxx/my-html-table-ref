my html table reference

table <tags>

tag name description
<table> table wrapper element for all HTML tables
<thead> table head set of rows defining the column headers in a table
<tbody> table body set of rows containing actual table data
<tfoot> table foot set of rows defining the footer in a table
<tr> table row table row container
<td> table data table data container



table attributes

attribute name description
colspan column span defines how many columns a td element should span.
rowspan row span defines how many rows a td element should span.
